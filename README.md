## Core Engineering

### [VoidBox](https://github.com/vishva2410/VOID-BOX) | Multi-Modal PII Detection & Redaction
Fully local, privacy-first AI engine designed to detect and redact 11+ PII categories with zero cloud dependency.
* **Architecture:** Hybrid YOLOv8 + EasyOCR + regex classification pipeline with LaMa neural inpainting.
* **Impact:** Achieves sub-2s/image inference on CPU with zero false-negatives across a 200-document test set.

### [Bodha AI](https://github.com/vishva2410/bodha_ai_1) | Real-Time Conversational Voice Assistant
Sub-300ms latency voice AI engine utilizing live audio streaming protocols.
* **Architecture:** LiveKit audio streaming integrated with WebSockets and NLP multi-turn context management.
* **Impact:** Delivered 95%+ conversation continuity in robust testing scenarios.

### [OncoDetect](https://github.com/vishva2410) | Modular Cancer Diagnostic Pipeline
Plug-and-play AI triage framework integrating organ-specific diagnostic models behind a unified interface.
* **Architecture:** Horizontally scalable; onboarding new organ modules requires zero core pipeline code modification.

### [DynaCalorie AI](https://github.com/vishva2410/Dyna-Calorie) | Adaptive Metabolic Engine
Metabolic tracking system that programmatically recalibrates TDEE using predicted vs. actual weight deltas.
* **Impact:** Outperforms static TDEE calculators by >12% accuracy over 12-week simulations.
